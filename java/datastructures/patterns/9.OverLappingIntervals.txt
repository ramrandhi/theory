⏰ Overlapping Intervals Pattern

Overlapping intervals problems often involve merging, inserting, or finding conflicts between intervals. The key insight is usually to sort intervals by start time and then process them in order to handle overlaps efficiently.

This pattern is widely used in:

    Scheduling
    Merging ranges
    Conflict detection

🧠 Pattern Recognition

| Pattern                   | Use When                                                |
| ------------------------- | ------------------------------------------------------- |
| Merge intervals           | Combine overlapping intervals into one                  |
| Insert interval           | Add an interval and merge if necessary                  |
| Meeting rooms / conflicts | Count overlaps to find required resources (e.g., rooms) |
| Interval intersections    | Find common overlaps between two interval lists         |
| Maximum overlap point     | Find the time point with max concurrent intervals       |

🧠 Bonus Concepts to Practice

| Concept               | Description                                                 |
| --------------------- | ----------------------------------------------------------- |
| Sorting intervals     | Always sort by start time (or end time for greedy picks)    |
| Two pointers          | Iterate two lists of intervals to find intersections        |
| Min-heap on end times | Track ongoing intervals for meeting room or resource counts |
| Sweep line algorithm  | Count start/end points to find max overlap or gaps          |

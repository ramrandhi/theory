| Use Case                                | Intuition                            |
| --------------------------------------- | ------------------------------------ |
| Sorted array?                           | Move inwards from both ends          |
| Need subarray?                          | Use sliding window with two pointers |
| Comparing values or finding partitions? | Move slow/fast pointers              |
| Remove/replace elements?                | Slow = writer, fast = scanner        |

Genarics: 
1. public class GenericClass<T> {} -> here T is a type parameter.
2. public <T> void method(T t) {} -> here T is a type parameter.

Wild Cards:
1. Upper Bound Wildcard: <? extends ClassName> 
   - This means that the type can be ClassName or any subclass of ClassName.
   - Example: List<? extends Number> can hold List<Integer>, List<Double>, etc.
2. Lower Bound Wildcard: <? super ClassName>
   - This means that the type can be ClassName or any superclass of ClassName.
   - Example: List<? super Integer> can hold List<Number>, List<Object>, etc.

3. Unbounded Wildcard: <?>
   - This means that the type can be any type.
   - Example: List<?> can hold List<Integer>, List<String>, etc.
4. Bounded Wildcard: <? extends ClassName & InterfaceName>
   - This means that the type can be ClassName or any subclass of ClassName and must implement InterfaceName.
   - Example: List<? extends Number & Comparable> can hold List<Integer>, List<Double>, etc. but also must implement Comparable.

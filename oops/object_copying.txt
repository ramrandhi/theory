SHALLOW COPY:
- A shallow copy of an object is a new object that is a copy of the original object, but it does not create copies of the objects that are referenced by the original object.
- In other words, a shallow copy creates a new object, but the fields of the new object are references to the same objects as the fields of the original object.
- This means that if the original object is modified, the shallow copy will also reflect those changes.
- Shallow copies are typically created using the clone() method or by using copy constructors that copy the values of the fields directly.

Example:
   class Person implements Cloneable {
      String name;
      Address address;

      Person(String name, Address address) {
         this.name = name;
         this.address = address;
      }

      public Object clone() throws CloneNotSupportedException {
         return super.clone(); // Shallow copy
      }
   }
   class Address {
      String city;

      Address(String city) {
         this.city = city;
      }
   }

DEEP COPY:
- A deep copy of an object is a new object that is a copy of the original object, and it creates copies of all the objects that are referenced by the original object.
- In other words, a deep copy creates a new object and also creates new objects for all the fields of the original object.
- This means that if the original object is modified, the deep copy will not reflect those changes.
- Deep copies are typically created using the clone() method or by using copy constructors that create new instances of the referenced objects.
- Deep copies are more expensive in terms of memory and performance compared to shallow copies, as they require creating new instances of all referenced objects.
- Deep copies are useful when you want to ensure that changes to the original object do not affect the copied object, especially when dealing with mutable objects.

Example:
   class Person implements Cloneable {
      String name;
      Address address;

      Person(String name, Address address) {
         this.name = name;
         this.address = address;
      }

      public Object clone() throws CloneNotSupportedException {
         Person cloned = (Person) super.clone(); // Shallow
         cloned.address = new Address(this.address.city); // Deep copy of nested object
         return cloned;
      }
   }
   class Address {
      String city;

      Address(String city) {
         this.city = city;
      }
   }

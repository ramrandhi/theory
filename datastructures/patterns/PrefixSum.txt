The Prefix Sum pattern is an essential technique to preprocess cumulative information, usually to answer range queries or perform efficient 
subarray computations in linear time.

🧠 When to Use Prefix Sum

| Scenario                        | Strategy                                                 |
| ------------------------------- | -------------------------------------------------------- |
| Fast range sum                  | Precompute prefix sum: `prefix[i] = sum of nums[0..i-1]` |
| Subarray sums                   | `prefix[j] - prefix[i] == k` → check using a HashMap     |
| 2D grid region sum              | Build 2D prefix matrix                                   |
| Balance checks (like 0/1 count) | Convert values to -1/+1, then use prefix sum             |
| XOR prefix trick                | For problems involving bitwise segment analysis          |

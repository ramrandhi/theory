🌐 DFS & BFS (Graph Traversal) Pattern

Depth-First Search (DFS) and Breadth-First Search (BFS) are fundamental algorithms for traversing or searching graph data 
structures. They’re widely used to explore nodes, find paths, detect cycles, and solve connectivity problems.

🧠 Pattern Recognition

| Pattern               | Use When                                      |
| --------------------- | --------------------------------------------- |
| DFS                   | Explore all possible paths, often recursively |
| BFS                   | Find shortest path in unweighted graphs       |
| Connected components  | Find all connected nodes or clusters          |
| Cycle detection       | Detect if a graph contains cycles             |
| Topological sorting   | Order nodes based on dependencies             |
| Level order traversal | BFS to process nodes level by level           |

🧠 Bonus Concepts to Practice

| Concept                    | Description                                              |
| -------------------------- | -------------------------------------------------------- |
| Recursive vs Iterative DFS | Recursive uses call stack; iterative uses explicit stack |
| Visited tracking           | Use boolean arrays or sets to avoid revisiting nodes     |
| BFS queue                  | Use queue for BFS to process nodes level by level        |
| Parent tracking            | For path reconstruction or cycle detection               |
| Topological sorting        | Use DFS post-order or BFS (Kahn’s algorithm)             |

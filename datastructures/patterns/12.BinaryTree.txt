🌳 Binary Tree Traversals Pattern

Tree traversals are fundamental for visiting all nodes in a binary tree. They help in solving problems related to 
searching, printing, or modifying trees.
Common traversal types:

    Preorder (Root, Left, Right)
    Inorder (Left, Root, Right)
    Postorder (Left, Right, Root)
    Level order (BFS)

🧠 Pattern Recognition

| Pattern               | Use When                                                   |
| --------------------- | ---------------------------------------------------------- |
| Preorder traversal    | You need to process the root before children               |
| Inorder traversal     | You need sorted order in BST or left-root-right processing |
| Postorder traversal   | You need to process children before root                   |
| Level order traversal | You need to process nodes level by level                   |

🧠 Bonus Concepts to Practice

| Concept                  | Description                                               |
| ------------------------ | --------------------------------------------------------- |
| Recursive traversal      | Easy to implement, natural for trees                      |
| Iterative traversal      | Use stack or queue to simulate recursion                  |
| Morris traversal         | Inorder traversal with O(1) space by threading tree nodes |
| Level order with markers | Use `null` or count nodes to separate levels              |

The Sliding Window pattern is a powerful technique to solve problems involving subarrays, substrings, or contiguous segments. 
It’s especially efficient when you need to:
    Find min/max/longest substring or subarray
    Count or sum within a range
    Track frequency/counts of elements

🧠 Pattern Recognition
| Pattern                | Use When                                                 |
| ---------------------- | -------------------------------------------------------- |
| Fixed-size window      | You know the exact size (e.g., size = `k`)               |
| Expand + shrink window | You don’t know the exact size, but must meet a condition |
| Track frequency/count  | Use HashMap or int[] to track window contents            |
| Window max/min         | Use Deque or Monotonic Queue for efficient tracking      |

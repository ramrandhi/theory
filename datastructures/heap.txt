HEAPS: 
Properties:
1. It actally stores data in an array but we need to visualize it in Binary Tree(Complete Binary Tree).
2. a. Every Node value is greaterThan or equal to (>=) all of its children. (max Heap)
   b. Every Node value is lessThan or equal to (<=) all of its children. (min Heap)
3. root => i = 1;
4. parent(i) = i/2;
5. left = 2*i;
6. right = 2*i + 1;
7. height is log(n);

1. MIN HEAP VS MAX HEAP

    Heap Type: Min-Heap  
    Property: Each parent ≤ its children

    Heap Type: Max-Heap  
    Property: Each parent ≥ its children

2.  Clarify Indexing  (1-based indexing)  (If using 0-based, these formulas change:)
    parent(i) = (i - 1) / 2
    left(i) = 2*i + 1
    right(i) = 2*i + 2

3. Add Downheap (Heapify-down / Sift-down) (Upheap is only half the story. For deletion (like poll() or remove()), Downheap is used.)
DOWNHEAP:
- Used when removing the root.
- Swap root with last element, then remove the last.
- Restore heap property by comparing with children and swapping down.

4. Mention Use Cases
USE CASES:
- Priority Queue
- Dijkstra’s Algorithm
- Heap Sort
- Task Scheduling
- Median Maintenance (using two heaps)

UPHEAP:
=> The upheap operation (also called bubble-up or heapify-up) is used in a heap (usually a binary heap) when you insert a new element 
and need to maintain the heap property.
=> we use ArrayList to make a priority Queue insted of insted of linkedlist because it takes O(n) where arraylist only takes O(log n).
=> What is Upheap?
    1. In a min-heap, the parent node must be less than or equal to its children.
    2. When you insert a new element, it is added at the end of the heap (i.e., the last position in the array).
    3. You then compare it with its parent:
    A. If the new element is smaller than its parent, you swap them.
    B. Repeat this process until the heap property is restored.

| Step          | Array Representation |
| ------------- | -------------------- |
| Before insert | `[1, 3, 6, 5, 9]`    |
| After insert  | `[1, 3, 6, 5, 9, 2]` |
| After upheap  | `[1, 3, 2, 5, 9, 6]` |

DATA STRUCTURES : 
 -> Java provides various types of data structures, which can be categorized into linear and non-linear data structures. Here’s an overview:

1. Linear Data Structures : 
  -> Linear data structures store elements sequentially.
  (a). Arrays  -> A fixed-size data structure that holds elements of the same type.
        int[] numbers = {1, 2, 3, 4, 5};

  (b). Linked List  ->  A dynamic data structure where elements (nodes) are connected through links.
        Types : 1. Singly Linked List, 2. Doubly LinkedList, 3. Circular Linked List.
        class Node {
            int data;
            Node next;
        }
        
   (c). Stack (LIFO - Last In, First Out)  ->  Uses push() to add elements and pop() to remove elements.
        Stack<Integer> stack = new Stack<>();
        stack.push(10);
        stack.pop();

    (d). Queue (FIFO - First In, First Out)  ->  Elements are added at the rear and removed from the front.
            Queue<Integer> queue = new LinkedList<>();
            queue.add(10);
            queue.poll();

    (e). Deque (Double-Ended Queue)  ->  Allows insertion and deletion from both ends.
            Deque<Integer> deque = new ArrayDeque<>();
            deque.addFirst(10);
            deque.addLast(20);

2. NON-LINEAR DATASTRUCTURES : 
  -> Non-linear structures store elements in hierarchical or interconnected ways.

  (a). Trees  ->  Hierarchical structure with nodes.
        1. Binary Tree, 2. Binary Search Tree, 3. AVL Tree, 4. Heap
        class Node {
            int data;
            Node left, right;
        }


  (b).  Graphs  ->  Collection of nodes (vertices) connected by edges.
        Representations : 1. Adjacency Matrix, 2. Adjacency List.
        class Graph {
            private LinkedList<Integer> adjList[];
        }           

        
   (c). (1).  Hash-Based Data Structures (HashMap)  ->  Stores key-value pairs.
                HashMap<String, Integer> map = new HashMap<>();
                map.put("Alice", 25);
                map.get("Alice");
        (2).  HashSet  ->  Implements a set (no duplicate elements).
                HashSet<Integer> set = new HashSet<>();
                set.add(10);
                set.add(20);

    (d). Queue (FIFO - First In, First Out)  ->  Elements are added at the rear and removed from the front.
            Queue<Integer> queue = new LinkedList<>();
            queue.add(10);
            queue.poll();

    (e). Deque (Double-Ended Queue)  ->  Allows insertion and deletion from both ends.
            Deque<Integer> deque = new ArrayDeque<>();
            deque.addFirst(10);
            deque.addLast(20);
